Animaciones CSS con Animations Web API

  • Botón agrandar: un 50% más durante 1s y luego volver a su estado normal
  • Botón rotar: 360 grados durante 1s y luego volver a su estado normal
  • Botón empequeñcer: un 50% menos durante 1s y luego volver a su estado normal
  • Se debe usar Animations Web API para todo, no CSS
