Animaciones CSS con JavaScript

  • Botón agrandar: un 50% más durante 1s y luego volver a su estado normal
  • Botón rotar: 360 grados durante 1s y luego volver a su estado normal
  • Botón empequeñcer: un 50% menos durante 1s y luego volver a su estado normal
  • Podemos crear un keyframe por cada animación,luego una clase para cada animación y luego aplicar la clase deseada con classList.add (antes debemos eliminarla con classList.remove y hacer un offsetWidth al elemento para que se pueda hacer más de una vez)
  • Otra opción es crear un keyframe por cada animación, luego usar style.animate para indicar la animación deseada (antes debemos poner animate a "" y hacer un offsetWidth al elemento para que se pueda hacer más de una vez)
